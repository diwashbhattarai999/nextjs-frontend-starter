
# Features

This folder contains the feature-focused modules for the app. Put feature-focused modules here, for example:

- `auth`
- `user`
- `billing`
- `example`

Each feature should be a self-contained module that can be used in the app.

## Auth Feature

The auth feature is a self-contained module that can be used to authenticate users. It contains the following modules:

- `login`
- `register`
- `logout`
- `forgot-password`
- `reset-password`
- `verify-email`
- `email-verification`
- `email-verification-success`

Each module should be a self-contained module that can be used in the app. It should be able to be used in the app without having to know about the other modules.
